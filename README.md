# Temperature Monitoring Application

## Introduction
This is a temperature monitoring application designed to track, log, and visualize temperature data from various sensors. It can be used in different environments such as homes, offices, and laboratories.

## Features
- Real-time temperature monitoring
- Historical data logging
- Configurable alerts for temperature thresholds
- Data visualization dashboards
- Integration with various temperature sensors

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/petrcimera-pixel/AI-Temp.git
   ```
2. Navigate to the project directory:
   ```
   cd AI-Temp
   ```
3. Install the required dependencies:
   ```
   npm install
   ```

## Usage
- To start the application, run:
  ```
  npm start
  ```
- Access the dashboard at `http://localhost:3000`

## Configuration
The configuration file can be found in the `/config` directory. You can customize various settings such as:
- Sensor types
- Alert thresholds
- Logging intervals

## Troubleshooting
- Ensure all sensors are correctly connected.
- Check the application logs for any errors.
